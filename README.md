# Ритм — офлайн-трекер привычек

Мобильное PWA-приложение без аккаунтов и сервера. Привычки и отметки хранятся только на устройстве в локальном хранилище браузера.

## Возможности

- добавление, изменение и удаление привычек;
- глобальное изменение названия, цели и цвета привычки;
- ежедневные отметки выполнения;
- календарь с переходом по месяцам и годам;
- цели на месяц, серии и общая статистика;
- работа без интернета после первой загрузки.

## Локальный запуск

Откройте папку проекта в любом локальном веб-сервере. Например:

```powershell
python -m http.server 4173
```

Затем откройте `http://localhost:4173`.

Для установки на телефон приложение должно быть однажды открыто по HTTPS. После этого выберите в браузере «Добавить на главный экран». Все пользовательские данные останутся только на телефоне.
